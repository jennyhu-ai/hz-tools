固定电价结算单生成（老模板）— 使用说明

更新日期：2026-05-28

功能概述

上传「电站基本信息」和「国网电费账单提取」两个 Excel 文件，按发电户号自动匹配数据，批量生成老模板样式的光伏电费结算单。支持单条/批量导出 Excel 和 PDF，内置 33 枚电子印章。

与新版的主要区别

  • 模板样式：老模板含 13% 税率拆分（税额、不含税金额）、人民币大写金额
  • 印章：内置 33 枚电子印章，无需手动上传
  • Excel 导出：包含「结算单」和「国网截图」两个工作表
  • 截图匹配：国网截图按账单文件名自动匹配

第一步：准备 Excel 文件

文件① 电站基本信息（.xlsx / .xls）

包含电站属性、电价、印章、银行账户等信息。第一行为表头，从第二行开始为数据。

文件② 国网电费账单提取（.xlsx / .xls）

需包含「账单汇总」工作表。若没有该工作表，系统自动读取第一个工作表。

第二步：上传与生成

  1. 点击或拖拽上传「电站基本信息」Excel
  2. 点击或拖拽上传「国网电费账单提取」Excel
  3. （可选）上传国网截图文件夹 — 图片名需与账单数据中的「文件名」列一致
  4. 系统自动按发电户号匹配，列表展示所有结算单记录
  5. 逐条操作：预览 / 下载 Excel / 下载 PDF
  6. 批量操作：下载全部 Excel / 一键打包 PDF (ZIP)
提示：所有文件上传均支持拖拽操作。印章已内置 33 枚，如提示缺少印章可补充上传。

结算单内容说明

  • 第1页：结算单主体
    • 业主单位、地址、发电户号、联系人及电话
    • 总发电量、总上网电量、总自用电量
    • 自用电折后电价、折后电费
    • 税率（13%）、税额、金额（不含税）、开票金额
    • 金额合计（大写）
    • 收款/付款单位账户信息
    • 印章、编制人、审核人、编制日期
    • 异议声明及联系方式
  • 第2页：国网截图附件（如有匹配截图）

电费计算公式

自用电折后电费 = 自用电量 × 固定电价 不含税金额 = 电费 ÷ 1.13 税额 = 电费 - 不含税金额 开票金额 = 自用电折后电费

国网截图匹配规则

  1. 账单 Excel 中「文件名」列记录对应的截图文件名（如 0dfcd186ae046249c2886c3a27c976e0.jpg）
  2. 上传截图文件夹后，系统按文件名（不含扩展名）进行匹配
  3. 匹配成功 → 截图显示在结算单第2页
  4. 匹配失败 → 工具栏显示「截图 X/Y」红色提示

常见问题

Q: 上传文件后匹配不到数据？

检查两个 Excel 的「发电户号」列是否一致（数字格式需统一），打开浏览器控制台查看日志输出。

Q: 印章不显示？

检查电站 Excel 中「印章图片」列的文件名是否与内置印章文件名完全一致（如 恒臻（铜陵）.png）。内置 33 枚印章列表可在控制台日志查看。

Q: PDF 一键打包很慢？

批量 PDF 使用 scale:3 高清输出，记录数量越多耗时越长。如只需快速预览可逐条下载。

Q: 下载的 Excel 中截图在哪？

Excel 文件包含两个工作表：底部标签可切换「结算单」和「国网截图」。
