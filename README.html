1. 约束单生成工具工具

批量生成约束单文档，支持图片导出与打印。

使用步骤

  1. 打开工具页面，上传相关数据文件
  2. 填写约束单参数（项目信息、约束内容等）
  3. 预览生成的约束单
  4. 逐条下载或批量导出为图片 / PDF
📄 文件：1-约束单生成工具/index.html

2. 国网电费账单数据提取工具

上传国网 PDF 账单文件，自动提取电量、电费等关键数据，并导出为 Excel 表格。

使用步骤

  1. 从网上国网下载电费账单 PDF 文件
  2. 上传 PDF 文件到工具页面
  3. 系统自动解析并提取账单数据
  4. 核对提取结果，导出为 Excel 文件

支持的数据字段

  • 发电户号、账单周期
  • 总发电量、上网电量、自用电量
  • 电费金额、电价信息
📄 文件：2-国网电费账单数据提取/index.html

3. 固定电价结算单生成（新）推荐

上传电站基本信息与国网账单数据，自动匹配并批量生成固定电价结算单。新版界面，内置电子印章，支持单条/批量导出 PDF 和 Excel。

使用步骤

  1. 上传「电站基本信息」Excel 文件（含电站属性、电价、印章图片名等）
  2. 上传「国网电费账单提取」Excel 文件（含账单汇总数据）
  3. 上传国网截图文件夹（图片名需与账单文件名对应）
  4. 系统自动按发电户号匹配数据，列表预览结算单
  5. 逐条预览/下载 Excel / PDF，或一键打包 ZIP

结算单包含内容

  • 业主单位、地址、发电户号、联系人
  • 总发电量、上网电量、自用电量
  • 自用电折后电价、折后电费
  • 收款/付款单位账户信息
  • 编制人、审核人、编制日期、电子印章
  • 第2页：国网截图附件
📄 文件：3-【新】固定电价结算单批量生成/index.html

4. 固定电价结算单生成（老）备用

旧版结算单生成工具，使用老模板样式（含税率/税额/不含税金额拆分）。内置 33 枚电子印章，支持截图匹配、多工作表 Excel 导出。

与新版的主要区别

  • 模板样式不同（老模板：含13%税率拆分、大写金额）
  • 内置 33 枚电子印章（无需手动上传）
  • Excel 导出包含「结算单」和「国网截图」两个工作表
  • 支持国网截图自动匹配（按账单文件名）

使用步骤

  1. 上传「电站基本信息」Excel（列名：发电户号、项目公司、业主单位、固定电价、印章图片 等）
  2. 上传「国网电费账单提取」Excel（含「账单汇总」工作表）
  3. 上传国网截图文件夹（可选，图片名与账单文件名对应）
  4. 列表预览、逐条下载 Excel / PDF，或一键打包 ZIP

电站基本信息 Excel 列名

序号 | 发电户号 | 项目公司 | 业主单位 | 电站简称 | 地址 | 业主联系人及电话 | 收款单位账户信息 | 付款单位账户信息 | 固定电价 | 印章图片 | 编制人 | 审核人 | 投资方联系人 | 投资方联系方式
注意：印章图片列填写印章文件名（如「恒臻（铜陵）.png」），需与内置印章文件名一致。
📄 文件：4-【老】固定电价结算单批量生成/index.html

5. 批量读取图片数据生成表格OCR

上传国网账单截图，通过 OCR 识别图片中的电量、电费等数据，批量导出为 Excel 表格。

使用步骤

  1. 准备国网账单截图（支持 PNG / JPG 格式）
  2. 批量上传图片到工具页面
  3. 系统自动 OCR 识别并提取数据
  4. 核对识别结果，修正识别错误
  5. 导出为 Excel 表格
OCR 识别依赖图片清晰度，建议使用高清截图以获得更准确的识别结果。
📄 文件：5-批量读取图片数据生成表格/index.html

通用注意事项

  • 所有工具均为纯前端运行，数据在浏览器本地处理，不会上传到服务器
  • 推荐使用 Chrome / Edge 浏览器以获得最佳兼容性
  • 批量生成 PDF 时请耐心等待，生成时间取决于记录数量
  • 如遇数据匹配失败，请检查 Excel 列名是否与工具要求一致
  • Excel 文件支持 .xlsx 和 .xls 格式
