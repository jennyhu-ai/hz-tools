约束单生成器 - 使用说明

用于生成施工现场违规约束单，支持图片导出与打印

功能概述

本工具用于施工现场管理，当发现安全、质量、文明施工等方面的违规行为时，可快速生成标准化的约束单文档，支持单张导出、批量导出图片以及直接打印。

操作步骤

1. 填写基本信息

在左侧编辑面板中依次填写以下字段，右侧会实时预览约束单效果：

字段说明必填
编号约束单唯一编号，如 YS-2026-001必填
工程名称当前施工项目名称必填
约束日期违规发生的日期必填
现场负责人水印及文档中显示的负责人姓名选填
施工班组对应合同中的施工班组名称选填
运营主管施工运营主管姓名选填

2. 设置违规信息

字段说明必填
违规类型下拉选择：安全违规 / 质量违规 / 文明施工违规 / 进度违规 / 材料管理违规 / 其他违规 / 客户满意度必填
违规程度一般违规 1元+ 或 严重违规 100元+必填
违规次数该人员/班组第几次违规，如"第1次"必填
乐捐金额罚款金额（元），如 200必填
约束事项详细描述违规内容和整改要求必填
违规依据下拉选择对应的违规依据条款必填

3. 上传违规图片

点击「违规图片」上传区域，选择现场拍摄的违规照片（支持 jpg、png 等常见格式）。图片将嵌入约束单中。

违规图片为必填项，导出前必须上传。

4. 导出 / 打印

所有必填项填写完整后，点击顶部按钮进行操作。如有缺失，系统会自动高亮红色并提示。

  • 导出当前图片：将当前约束单导出为 PNG 图片
  • 批量导出图片：结合批量模板 CSV 文件，一次性生成多张约束单图片
  • 打印：直接调用浏览器打印功能输出纸质约束单
  • 清空：重置所有已填写的表单内容

批量导出说明

点击「批量导出图片」按钮前，需准备一个 CSV 模板文件（参考 批量模板_测试数据.csv），包含多条记录，每条对应一张约束单。工具将逐条渲染并打包下载。
