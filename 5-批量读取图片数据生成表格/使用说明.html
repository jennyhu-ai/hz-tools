批量读取图片数据生成表格 - 使用说明

上传国网电费账单截图，OCR 自动识别关键数据并导出 Excel

功能概述

本工具用于批量处理国网（国家电网）电费账单的手机截图，通过 OCR 文字识别技术自动提取账单中的关键字段，汇总后导出为 Excel 表格，方便结算对账。

操作步骤

1. 上传图片

点击上传区域或拖拽图片到虚线框内，支持 JPG / PNG / WebP 格式，可一次选择多张截图。

2. 开始识别

点击「开始识别」按钮，系统将逐张读取图片中的文字并提取数据。首次使用需下载中文语言包（约 10MB），后续自动缓存。

3. 查看结果

识别完成后，结果表格展示所有提取到的字段数据，底部显示汇总统计。

4. 导出 Excel

点击「下载 Excel」即可导出包含所有数据的表格文件。

适用截图类型

适用于国网 App（网上国网）中「电费账单」页面的手机截图，包含电量电费明细信息。

截图需清晰完整，避免模糊、反光或遮挡。建议使用原始分辨率截图，不要过度压缩。

常见问题

Q: 识别结果为空或不准确？

点击文件名旁的「查看识别原文」可展开 OCR 原始输出文字，检查是否正确识别了中文字符。如关键字段缺失，可能是截图不清晰或格式不匹配。

Q: 首次加载较慢？

首次使用需从 CDN 下载 Tesseract 中文语言包（约 10MB），后续浏览器会缓存，加载速度会明显提升。

Q: 支持哪些浏览器？

推荐使用 Chrome / Edge 等现代浏览器，需支持 Web Worker 和 Canvas API。

技术说明

OCR 引擎：Tesseract.js v5（chi_sim 中文识别）

Excel 导出：SheetJS（xlsx）

所有处理均在浏览器本地完成，图片不会上传到任何服务器。
